# 📘 WorkTimeTrayTool – Guida all’Installazione e all’Uso

Questa guida descrive come installare e avviare automaticamente **WorkTimeTrayTool**, oltre a spiegare dove trovarlo una volta in esecuzione.

---

## 📂 Installazione

### 1️⃣ Scarica l’ultima versione
Apri la pagina delle release:

👉 **https://github.com/WorkTimeTrayTool/WorkTimeTrayTool-Releases/tree/main/releases**

Scarica il file **ZIP** relativo all’ultima versione disponibile.
<img width="1822" height="354" alt="image" src="https://github.com/user-attachments/assets/801b0b9d-ed4d-4c58-9795-486feff630de" />

---

### 2️⃣ Estrai la cartella
Una volta scaricato lo ZIP:
- estrai il contenuto;
- copia la cartella **WorkTimeTrayTool** in una posizione locale a tua scelta.

Esempi consigliati:
- `C:\Program Files\WorkTimeTrayTool`
- `E:\WorkTimeTrayTool`
<img width="860" height="191" alt="image" src="https://github.com/user-attachments/assets/e49dc9f2-e56c-40e9-a089-02064f2f6e5f" />


---

## 🔄 Aggiornamento

Se possiedi già una versione precedente dell’app:

👉 **sostituisci soltanto il file `WorkTimeTrayTool.exe`** nella cartella esistente.

Non serve riconfigurare nulla.

---

## 🚀 Avvio Automatico con Windows

Per configurare l’avvio automatico dell’applicazione:

1. Premi **Win + R**
2. Digita il comando: *shell:startup*
3. Questo apre la cartella *Startup* di Windows.
3. Crea un **collegamento (shortcut)** all’eseguibile `WorkTimeTrayTool.exe`
4. Copia lo shortcut nella cartella *Startup* aperta al punto 2.

<img width="1432" height="598" alt="image" src="https://github.com/user-attachments/assets/2e4824c6-14ba-4383-80a3-3f85acc5c6c0" />



Da ora in poi l’applicazione verrà avviata automaticamente all’accensione del PC.

## ℹ️ Note

- Assicurati che il collegamento punti alla cartella locale in cui hai copiato l’app.
- Se modifichi la posizione della cartella, aggiorna anche lo shortcut nella cartella *Startup*.
- Nella cartella verrano automaticamente create delle cartelle come : *Logs* e *Output*, **se le eliminarai verranno ricreate ma il loro contenuto sarà perso.**
